# VT-Claw 

vt-claw 是一款以硬件技能（Skills）为核心，面向行业用户的 Claw 框架，优先集成硬件相关技能，通过灵活且安全的开源技术架构，支持各行各业的物理智能体开发与定制，提供主动智能、透明可控、安全隔离的运行底座。

## 主要创新

- 主动智能：通过自研的 [SKILL.mk](https://github.com/Teaonly/SKILL.mk) 技术 ，提供主动演化改进、精确审计的外部技能。
- 透明可控：基于 Pi-Mono 定制一整套的 Harness 外挂，实现透明的上下文工程，为行业应用提供稳定评测和复现路径。
- 安全隔离：严格的容器环境和外部资源配置，满足行业应用基础安全要求。
