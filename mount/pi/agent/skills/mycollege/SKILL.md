---
name: mycollege
description: 查询我所在单位（我的工作单位是一所大学）的信息服务，包括站内信，课程表，以及学校通知。
---

## Commands

### 查询站内信

```bash
opencli mycollege msg [limit] # 查询站内信，默认返回最近10条
```

### 查询校内通知

```bash
opencli mycollege news [limit] # 查询校内通知，默认返回最近11条
```

### 查询课程表

```bash
opencli mycollege course  # 查询当前学习课程表
```
